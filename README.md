# Secret Santa

Secret santa is among the most fun end of year activities that many groups of individuals participate in.
This app provides a quick and easy way to set up a secret santa and have it be anonomyous and fair for everyone.

## Features

 - Friends list functionality
 - Create a new secret santa event + Add friends to it
 - Have a unique block list per person (to prevent couples from purchasing from one another)
 - randomly assign purchasee
 - view all upcoming events

## Missing/Planned features

 - allow users to drop from an event and handle the "who's buying for who" update
 - Proper login functionality
 - e-mailing users once the event begins
 - Duplicate existing event from previous year (same people, same block list, new event)
